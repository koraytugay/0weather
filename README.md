# 0weather

A minimal, mobile-first weather app for Toronto. Vanilla HTML, CSS, and JS — no frameworks, no build step.

Live at: https://koraytugay.github.io/0weather

## Features

- Current temperature, feels like, condition, daily range, and wind
- Hourly forecast cards for today and tomorrow (swipeable)
- Smart notes: wind changes, rain/snow start and stop times
- Significant weather highlighting (high wind, extreme cold)

## Data

Weather data from the [Open-Meteo API](https://open-meteo.com/) — no API key required.
